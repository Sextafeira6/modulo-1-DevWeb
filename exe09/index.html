Usando hiperlinkagem e Ancoragem

Para criar um hyperlink, devemos criar âncoras através da tag <a>

O principal atributo dessa tag é o href, que cria uma referência hipertexto

1) links Externos

Links Externos levam o leitor da sua página para uma outra página de outro site que não o seu domínio

Exemplo: vamos visitar o primeiro site do mundo

Atenção: por padrão,o hiperlink abrirá um site no lugar do frame que está aberto o seu site

  • No entanto, no caso de direiconamento para links externos, o recomendado é abrir uma nova aba
    • Para isso basta usar o atributo: target="_blank"
      • Ex: vamos visitar a 1ª TLD ".com" registrada na história de internet

Ah! além disso é bom demonstrar para os motores de busca que a ancoragem que você está usando não é relacionada ao seu domínio, isto é, ela é externa ao seu site.

  • Para isso basta usar o atributo rel="external":

Links Dofollow e Nofollow

Links Dofollow:
Permitem que os motores de busca apontem de volta o seu site (atributo rel="dofollow")
Links Nofollow
São aqueles que você pode inserir e não ter um apontamento de volta (atributo rel="nofollow")

🤔Dúvida: Qual a utilidade de diferenciar links Dofollow de Nofollow?

🧠Resposta[...]da mesma forma que links bons apontando o seu domínio o tornam melhor, links ruins podem diminuir a sua autoridade diante do Google.

O uso adequado dos atributos external, nofollow ou dofollow depende de uma pesquisa mais apurada sobre SEO

2) Links Internos

Links internos permite o direcionamento do leitor da sua página para uma outra página sua dentro do seu site (domínio)

Por exemplo você já visitou a minha segunda página HTML? vamos lá?


Detalhes importantes

  • Se o arquivo da página referida que você quer mandar o leitor estiver no mesmo local (pasta) que está sua página referente dá pra colocar apenas o nome do arquivo e blz
  • Se o arquivo da página referida estiver dentro de uma pasta que está dentro da onde está sua página referente, nesse caso você vai ter que usar no link o sufixo: nome da pasta/nome do arquivo
  • Se o arquivo da página referida estiver fora da pasta onde está a página referente nesse caso você vai ter que usar o sufixo: .../nome do arquivo

Exemplos abaixo

  • Ir para página 02 (estrutura: <a href="pg2teste.html" >Ir para página 02</a> )
  • Ir para página 03 (estrutura: <a href="pg3teste/pg3teste.html">Ir para página 03 </a>)</li>
