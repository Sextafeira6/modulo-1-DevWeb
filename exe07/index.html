1.1) Pré formatação

Agora para você manter uma formatação "profissional" de uma linha de código, como essa aqui embaixo ⬇ ,use a tag <pre> (se não, as linhas ficam lado a lado)

                
                Num = int(input('Digite um número'))
                if num % 2 == 0:
                print(f'0 número {num} é PAR)
                ELSE:
                print(f'O número {num} é ÍMPAR')
                print('Fim do programa')

😉Dica extra: para escolher em qual parte horizontal o código vai ficar, selecione apenas o texto do código (sem as tags) e use os comandos do teclado:

▶TAB para afastar no sentido direito

◀Shift + TAB para trazer no sentido esquerdo

Exemplo:◀

             
Num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'0 número {num} é PAR)
ELSE:
    print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

2) Citações

2.1) Citações Simples

Como diria um velho sábio (Einstein): Os computadores são incrivelmente rápidos, precisos e burros; os homens são incrivelmente lentos, imprecisos e brilhantes; juntos, seus poderes ultrapassam os limites da imaginação.

Para fazer citações, como essa ai em cima, use a tag <q> pois é uma tag puramente semântica que indica para a linguagem que trata-se realmente de uma citação, pois deixando entre aspas, o html iria tratar apenas como um termo integrante do texto.


2.2) Citações compostas

Descloada um pouco para direita, de forma a dar ênfase e destaque para a parte citada além de ter uma semântica própria para o HTML

😉Dica extra: se você quiser deixar as devidas referências ao site de onde você pegou essa citação (para não violar algum direito autoral), você pode dentro da tag <blockquote> colocar um link de citação para o site de onde você pegou a parte do texto citado usando o parâmetro cite="link do site" dentro daquela tag, no caso vai ficar assim <blockquote cite="https:// cite citado ..."> citação </blockquote>. Abreviações e Conceitos suspensos

A tag ➡abbr⬅ é uma "caixa suspensa" que mostra uma definição daquilo que você resolveu conceituar ou abreviar, assim você ajuda tanto os mecanismos de busca, como simplifica seu texto, ah a propósito, você sabe o que é um motor de busca né?!
