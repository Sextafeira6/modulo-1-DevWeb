Trabalhando com listas

1) Listas Ordenadas

Para criar uma lista ordenada siga a sequência:

  1. Abra a tag <ol>
  2. Quebre a linha (por organização)
  3. Liste as coisas uma a uma por meio da tag <li>
  4. É isso!

1.1) Tipos de início da lista

Ah você pode também modificar qual o tipo de ordenamento da lista, por exemplo, se é por números ordinais, romanos, alfabeto.

Para isso basta usar o parâmetro type depois da abertura da tag <ol>, ele fornece os seguintes tipos de ordenações: (fica assim <ol type="">)

  1. 1 - lista numérica comum
  2. A - Lista alfabética - Caixa alta
  3. a - Lista alfabética - Caixa baixa
  4. I - Lista numérica - Algarismos romanos - Caixa alta
  5. i - Lista numérica - Algarismos romanos - Caixa baixa

Exemplo:

  1. Entendeu?
  2. Agora a gente vai ver como começar uma lista a partir de um outro número/ letra qualquer
  3. Por exemplo, começar uma lista a partir do 5

1.2) Iniciando uma lista a partir de um referêncial específico

Para iniciar a lista a partir de um referêncial específico basta usar o parâmetro start (tipo assim: <ol type="1" start="5">)

  1. A
  2. cor
  3. da
  4. Pedrinho
  5. que hoje tem campeonato!!!

⚠ Atenção ⚠: O parâmetro start usa valores numéricos, incusive nas listas alfabéticas

Exemplo: se você quiser começar a partir da letra "c", o parâmetro start vai ter ter um valor: 3

2) Listas não ordenadas

Eu também posso criar listas que não são necessariamente sequênciais, isto é, tem um padrão, tipo 1, 2, 3 ... são topics points por exemplo:

  • O controle de constitucionalidade pode ser difuso ou concentrado
  • Ele mata a cobra e mostra o pau
  • Eu vou a diante e volto atrás, por isso do lugar não saio mais

2.1) Tipos de lista não ordenadas

Para colocar outros tipos de points na lista é preciso usar o parâmetro type do mesmo modo como lá em cima, aqui existem os seguintes tipos de points:

  1. Disc ● (padrão)
  2. Circle ○
  3. Square ■

3) Listas mistas

Para criar listas mistas, basta colocar um <ol> dentro de outra <ol> ai vai da sua criatividade! exemplo abaixo

  1. Linguagens de marcação
    1. HTML
    2. XHTML
    3. SGML
  2. Linguagens de programação
    1. Python
    2. JavaScript
    3. Java

💡Dica extra💡: Se quiser selecionar/ alterar várias tags ao mesmo tempo, faça o seguinte:

  1. Coloque o ponto de Inserção {PI} na parte que você quer alterar da tag
  2. Para cada tag que você for colocar o PI, antes, aperte a tecla Alt
  3. Edite

⚠Ah! dois detalhes ⚠:

  • Onde vc deixar o PI (dentro da tag) é a parte que vai ser editada!
  • Todos os PIs serão editados pelo mesmo termo que escrever

4) Lista de Definição

Para criar uma lista de definição

  1. Abra a tag <dl>
  2. Digite o Termo a se conceituar dentro da tag <dl>
  3. Qubre a linha (por organização)
  4. Abra a tag <dd>
  5. Digite dentro da tag <dd> a definição do conceito que está contido na tag <dl>
